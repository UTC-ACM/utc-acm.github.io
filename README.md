# How does the site work?
This site was built using the Jekyll static site generator, which was written in Ruby.

GitHub Pages is powered by Jekyll, so support is built in.

Before making changes to the site, you will need to install Jekyll and its dependencies. See [this page](https://jekyllrb.com/) for more information.
